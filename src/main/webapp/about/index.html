SMOOTH - первый сервис построения маршрутов по городу с обходом препятствий

Мы все имеем различные физические возмжности, но наш город должен быть доступным для каждого. SMOOTH позволяет путешествовать по городу людям разной степени мобильности. Каждый человек может добавить информацию о препятствии на его пути. В дальнейшем это обязательно поможет кому-то безопасно достичь своей цели.

Видео презентация

SMOOTH (iOS, Android, Windows Phone)

Сервис Smooth доступен на всех современных смартфонах в виде вэб-приложения. Вам достаточно лишь открыть ссылку в браузере и построить маршрут.

Возможности

Состояние тротуаров

Благодаря открытым данным и нашим пользователям Smooth знает о проблемах на пешеходных путях и находит доступный маршрут.

Уклоны, ступеньки, поребрики

Отсутствует спуск на пешеходном переходе, ступеньки на тротуаре, слишком крутой уклон. Smooth избавит от риска оказаться в тупике.

Оборудованность транспорта

Smooth найдёт ближайший транспорт с низким полом. При прокладке маршрута Smooth предлагает только доступный транспорт.

Наша суперкоманда ищет волонтёров!

Сервис Smooth позволяет людям, для которых прогулка по улицам достаточно рискованное мероприятие, выходить из дома не опасаясь оказаться в тупике или опаздать на 3 часа из-за отсутствия транспорта. Но предлагать действительно лучшие и самые надёжные маршруты мы сможем только с вашей помощью.

Вы можете помочь внося вклад в покрытие пешеходных путей на карте Санкт-Петербурга в OpenStreetMap, оставляя данные о препятствиях через наше веб-приложение или предлагая функции, которых по вашему мнению не хватает Smooth.

Сервис Smooth стал возможен благодаря потрясающим людям и организациям, которые ежедневно вносят вклад в развитие открытых данных и повышают их качество. Вливайтесь!
